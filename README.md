# ```Television-Show-Search```
```A simple project showcasing how to manipulate an API in Sveltejs```

A simple [Svelte.js](https://svelte.dev/) project that demonstrates how to manipulate an API to create a dynamic search feature for television shows. This project showcases the use of modern JavaScript frameworks to fetch and display data from external APIs, providing a user-friendly interface for searching and viewing TV show details.
